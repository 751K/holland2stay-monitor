本隐私政策说明 FlatRadar 如何收集、使用、存储和保护您的数据。

FlatRadar 是一款独立的、非官方的租房监控工具，跨平台跟踪 Holland2Stay、OurDomain、Xior 等多个平台的房源。FlatRadar 与其监控的任何房源平台均不存在关联、认可、赞助、维护或运营关系。

使用 FlatRadar，即表示您确认已阅读并理解本隐私政策。

2. 我们收集的数据

根据您使用 FlatRadar 的方式，我们可能会收集和处理以下类别的数据：

账户数据
- 用于创建 FlatRadar 账户的电子邮件地址或用户名。
- 您 FlatRadar 账户的密码认证数据。
- 账户创建日期、登录状态、账户设置和账户删除状态。
- 用户角色或账户类型（如适用）。

密码存储

FlatRadar 账户密码不以明文形式存储。密码使用 bcrypt（一种单向密码哈希机制）进行保护。这意味着 FlatRadar 存储的是密码验证数据，而非您原始的明文密码。

认证与会话数据
- 用于保持登录状态的会话令牌、刷新令牌或类似的登录令牌。
- 设备特定的认证状态。
- 登录时间戳和与安全相关的认证记录。
- 失败的登录尝试或其他安全事件（在保护服务所必需时）。

设备与通知数据
- Apple 推送通知服务设备令牌。
- 设备类型、应用程序版本、操作系统版本和通知设置（在应用程序功能所需时）。
- 用于维护和调试服务的基本设备或应用程序诊断信息。

生物识别认证数据
- FlatRadar 支持 Face ID 和 Touch ID 以便捷登录。
- 您的生物识别数据（面部扫描、指纹）完全由 Apple 的 LocalAuthentication 框架在您的设备上处理。FlatRadar 从不接收、存储或访问您的生物识别数据。
- 当您选择保存登录信息以使用 Face ID 或 Touch ID 时，您的账户凭据（用户名和密码）将本地存储在您设备的 Keychain 中，并受生物识别认证保护。FlatRadar 不会上传这些本地存储的凭据。

位置数据
- 如果您使用地图定位按钮，您的设备可能会向应用程序提供您的当前位置，以便地图以您为中心显示。
- FlatRadar 仅在您的设备上本地使用此位置信息以提供地图功能。
- FlatRadar 不会上传、存储、保留或将您的精确设备位置与您的账户关联。

已保存的偏好设置与提醒数据
- 已保存的筛选条件、城市、价格范围、可用性、房源类型、房源状态及其他搜索偏好设置。
- 通知偏好设置。
- 提醒历史记录，包括匹配的房源、提醒时间戳以及操作通知所需的送达状态。

反馈数据
- 当您通过应用程序提交反馈或报告错误时，我们会收集您的反馈类型（错误报告 / 建议 / 其他）、消息内容、账户标识符、用户名、应用程序版本和设备平台。
- 反馈数据用于理解、调查和改进 FlatRadar。

使用、网络和技术数据
- 基本的应用程序使用事件、服务器日志、请求时间戳、错误日志、IP 地址、设备信息、应用程序版本和诊断信息。
- 用于防止滥用、垃圾信息、未经授权访问、抓取滥用或服务中断的安全相关日志。

3. 我们不主动收集的数据

FlatRadar 不主动收集敏感个人数据，如种族、宗教、政治观点、健康信息、生物识别数据或精确的实时位置。地图功能使用的设备位置在您的设备上本地处理，FlatRadar 不予存储。

FlatRadar 不出售您的个人数据。

FlatRadar 不将您的个人数据用于第三方广告或跨应用追踪。

4. 我们如何使用您的数据

我们将您的数据用于：创建和管理您的账户、认证您的身份、保存您的筛选条件和偏好设置、为匹配的房源发送推送通知、记录提醒历史、显示房源信息、同步数据、运营和保护服务、调试和维护服务、防止滥用以及遵守法律义务。

5. 数据处理的法律依据

在 GDPR 或类似数据保护法律适用的情况下，我们基于以下法律依据处理您的数据：履行合同、同意（针对推送通知等可选功能）、合法利益（安全、维护、防止滥用）以及在必要时履行法律义务。

6. 数据存储

FlatRadar 将数据存储在您的设备上（应用程序偏好设置、本地缓存、认证状态、Keychain 凭据）以及 FlatRadar 服务器上（账户数据、bcrypt 哈希密码、会话令牌、已保存的筛选条件、推送令牌、提醒历史记录、服务器日志和安全日志）。FlatRadar 可能会使用第三方基础设施提供商来运营服务。

密码不以明文形式存储在 FlatRadar 服务器上。

7. Cloudflare

FlatRadar 使用 Cloudflare 提供 DNS、流量路由、安全、性能及相关基础设施服务。Cloudflare 可能会根据其自身的隐私政策处理技术数据，例如 IP 地址、请求标头和安全事件。

8. 推送通知

FlatRadar 使用 Apple 推送通知服务发送房源提醒。我们存储您的 APNs 设备令牌并将其与您的账户关联。您可以随时在 iOS 设置中禁用推送通知。

9. Holland2Stay 与房源数据

FlatRadar 可能会从公开可访问的 Holland2Stay 服务、API 或技术接口获取房源信息。Holland2Stay 独立于 FlatRadar。FlatRadar 不控制 Holland2Stay 的数据、系统、条款或隐私实践。

10. 第三方服务

FlatRadar 可能会与以下服务交互：Holland2Stay（房源数据）、Apple 推送通知服务（通知）、Cloudflare（DNS 和安全）、托管和后端提供商以及电子邮件或支持服务提供商。

11. 数据共享

我们不出售您的个人数据。我们不将您的个人数据用于第三方广告或跨应用追踪。我们仅在运营 FlatRadar 所必需的情况下与服务提供商共享有限数据。如果法律要求或为保护权利和安全，我们可能会披露信息。

12. 数据保留

我们仅在必要时保留您的数据。账户数据在您的账户处于活跃状态期间保留。服务器日志和诊断数据保留有限期限。当您删除账户时，我们将删除或匿名化关联的个人数据，除非法律、安全或合法运营需要保留。

您可以通过联系 {{CONTACT_EMAIL}} 请求删除您的账户及相关数据。删除后，您可能会失去对已保存的筛选条件、通知设置、提醒历史和其他基于账户的功能的访问权限。

15. 安全

我们采用合理的技术和组织措施保护您的数据，包括 HTTPS、服务器端访问控制、bcrypt 密码哈希、安全的会话处理、Cloudflare 安全保护、Apple Keychain（如适用）以及用于检测滥用的监控。FlatRadar 账户密码不以明文形式存储。
